# Bluesky feed for Statamic websites

## Fetch posts from a Bluesky account, customise and display them on your Statamic website

### [View live demo &rarr;](https://vhbelvadi.com/bluesky-feed)

**Want to jump in the deep end?** Install this add-on via Composer and get started:

``` bash
composer require vhbelvadi/bluesky-feed
```
